# ldbb
`ldbb` is a Python script written by Aaron Webster for calculating optical constants of several metals using the Drude, Lorentz-Drude, or Brendel-Bormann models. The code is based on the paper by A. D. RakiÄ‡ et al. "Optical properties of metallic films for vertical-cavity optoelectronic devices", [Appl. Opt. 37, 5271-5283 (1998)](http://dx.doi.org/10.1364/AO.37.005271)

## Files in this directory:

`ldbb.py` - Returns the complex permittivity for different metals using the Drude, Lorentz-Drude, or Brendel-Bormann models.

`example.py` - An example how to use ldbb.py

`example2.py` - Enother example how to use ldbb.py
